# 📖 About TravelBharat

TravelBharat is a modern tourism platform designed to help people explore India's incredible diversity through an immersive and responsive web experience.

Users can discover **States**, **Union Territories**, and famous **Tourist Destinations** with detailed information, image galleries, and direct Google Maps navigation.

The project focuses on providing a clean UI, smooth animations, and an intuitive user experience.

---

# ✨ Features

### 🌍 Explore India

- 🇮🇳 28 Indian States
